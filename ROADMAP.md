# Benetto Roadmap

## v1.0 — Current (May 2026) ✅
- [x] Voice recording with Android MediaRecorder
- [x] On-device transcription via Whisper.cpp Small
- [x] Room database for notes & transcripts
- [x] Basic playback of recordings
- [x] Material 3 UI with Jetpack Compose
- [x] Russian & English language support
- [x] APK distribution via website

## v1.1 — Stability & Polish (June 2026)
- [ ] Background recording (foreground service)
- [ ] Notification controls (pause/resume/stop)
- [ ] Batch delete & selection mode
- [ ] Recording quality settings (bitrate, format)
- [ ] App tour / onboarding screens

## v1.2 — Smart Features (Q3 2026)
- [ ] Transcription search (full-text)
- [ ] Speaker diarization (multi-speaker detection)
- [ ] Export transcripts (TXT, SRT, JSON)
- [ ] Tags & categories for notes
- [ ] Dark mode improvements

## v2.0 — Advanced (Q4 2026)
- [ ] Real-time transcription (streaming mode)
- [ ] Multiple Whisper models (Tiny, Base, Small, Medium)
- [ ] Model download manager (in-app)
- [ ] Audio trimming & editing
- [ ] Cloud backup (optional, encrypted, user-initiated only)
- [ ] Google Play Store release

## Future Ideas
- [ ] Translation of transcripts
- [ ] Meeting summarization (on-device LLM)
- [ ] iOS version (Swift + WhisperKit)
- [ ] Wear OS companion app
- [ ] Shared note collaboration (P2P)
